# Senior Editorial Theme

A high-authority, agency-tier theme for senior software engineering leaders. It balances a warm, physical substrate with sharp technical precision.

## Color Palette

- **Substrate**: `#FDFBF7` - Primary page background (warm unbleached paper)
- **Ink**: `#111111` - Primary text and bold headings
- **Hazard Red**: `#FF2A2A` - Singular high-contrast accent for focus and alerts
- **Surface**: `#FFFFFF` - Card and modal backgrounds for clean layering
- **Border**: `#E4E4E7` - Fine hairline dividers (Zinc-200)
- **Muted**: `#F1F1EF` - Substrate-toned neutral for background elements
- **Caption**: `#71717A` - Secondary metadata and placeholders (Zinc-500)

## Utility Colors (Desaturated)

- **Success**: `#10B981` - Desaturated Emerald for safe states
- **Info**: `#0EA5E9` - Desaturated Sky for neutral signals
- **Warning**: `#F59E0B` - Desaturated Amber for cautionary status
- **Danger**: `#EF4444` - Refined red for destructive actions

## Typography

- **Headers**: Cabinet Grotesk (Black)
- **Body**: Satoshi (Medium)
- **Technical**: JetBrains Mono (Medium)

## Best Used For

Senior Engineering Portfolios, Strategic Architecture Docs, Premium Technical Blogs.
